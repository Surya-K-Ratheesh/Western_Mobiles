## ✨ Overview

**Western** is a premium phone shop landing page featuring an immersive **3D interactive experience** built with Three.js. It showcases the store's services with a modern, futuristic aesthetic.

![Preview](https://img.shields.io/badge/3D-Three.js-gold?style=flat-square)
![Stack](https://img.shields.io/badge/Stack-Vanilla%20JS-ffd700?style=flat-square)
![Hosting](https://img.shields.io/badge/Hosting-Vercel-black?style=flat-square)

---

## 🚀 Features

- **3D Product Showcase** — Floating phones rotate in 3D space with real-time particle effects
- **Interactive Camera** — Mouse/touch movement controls the 3D camera perspective
- **Responsive Design** — Works seamlessly on desktop and mobile
- **Services Grid** — Highlights phone repair, watches, clocks, and accessories
- **One-Click Contact** — Direct call button and Google Maps integration

## 🧰 Tech Stack

- **Three.js** — 3D rendering engine

- **CSS3** — Animations, gradients, responsive layout
